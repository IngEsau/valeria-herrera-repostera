# Landing Home Edge Cases

## Visuales
- Hero demasiado decorativo y poco legible
- CTA sin contraste suficiente
- Exceso de texto en “Acerca de mí”
- Imágenes de producto con composición inconsistente

## Responsive
- Header colapsado incorrectamente en mobile
- CTA principal desplazado demasiado abajo
- Cards de producto demasiado altas o cortadas
- Footer saturado en pantallas pequeñas

## Contenido
- Precio no disponible
- Producto sin descripción
- Nombre de producto demasiado largo
- Copy de hero demasiado corto o demasiado extenso

## Conversión
- Canal principal no definido
- Múltiples CTAs compitiendo
- CTA visualmente fuerte pero sin destino claro
