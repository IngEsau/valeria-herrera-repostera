# Landing Home Test Plan

## Objetivo
Validar que la landing principal cumpla criterios de claridad, consistencia visual, responsive y orientación a conversión.

## Casos principales
- El hero comunica la propuesta de valor
- El CTA principal es visible y claro
- La navegación principal no distrae
- La sección “Acerca de mí” refuerza confianza
- Los productos destacados se entienden rápido
- El footer mantiene cierre visual consistente

## Casos responsive
- mobile portrait
- mobile landscape [si aplica]
- tablet
- desktop
- pantallas amplias

## Casos de accesibilidad
- contraste entre texto y fondo
- legibilidad del CTA principal
- jerarquía de headings
- enlaces reconocibles
- orden visual coherente

## Casos negativos
- imagen faltante en producto
- texto excesivamente largo en producto
- CTA sin destino operativo
- contenido insuficiente en una sección
- assets visuales de baja calidad

## Evidencia requerida
- screenshots por breakpoint
- checklist de criterios
- observaciones de legibilidad
- validación manual de flujo visual
