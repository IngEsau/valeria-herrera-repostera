# Domain Entities

## 1. Brand
### Descripción
Representa la identidad principal del sitio.

### Atributos iniciales
- nombre
- descriptor
- logo principal
- isotipo
- firma secundaria
- tono de marca
- principios visuales

## 2. Landing Section
### Descripción
Bloque funcional de la SPA.

### Tipos previstos
- header
- hero
- about
- featured-products
- testimonials
- footer
- cta-strip [si se valida]

## 3. Featured Product
### Descripción
Postre mostrado con intención comercial dentro de la landing.

### Atributos iniciales
- nombre
- descripción breve
- imagen
- precio [validar si siempre aplica]
- CTA asociado

## 4. CTA
### Descripción
Elemento de acción orientado a conversión.

### Atributos iniciales
- label
- destino
- prioridad visual
- contexto de uso

## 5. Contact Channel
### Descripción
Canal de contacto o conversión.

### Estado
No definido. Debe validarse si será WhatsApp, formulario, Instagram, teléfono u otro canal.
